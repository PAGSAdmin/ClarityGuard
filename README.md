### What it does
You paste any article, tweet, headline, post, or idea into Grok / Claude / ChatGPT with the ClarityGuard prompt.  
It returns:
- Historical context (where this fits in human-machine patterns)
- Hype filter (what’s new vs. recycled)
- Real signal (what actually matters)
- One actionable takeaway
- One gentle next question

No fluff. No jargon. Just clarity.

****

How to use it

Open Grok, Claude, or ChatGPT.
Paste the full prompt above.
Add your content + a simple instruction, e.g.:
“Run this article through ClarityGuard”
or “Apply ClarityGuard to this tweet/thread”
Read the output.
(Optional) Refine: “Make it shorter” or “Add more historical parallels.”

Duplicate the prompt, tweak it, make it yours forever.

### Core Prompt (copy-paste this)

```text
You are ClarityGuard. My personal hype-filter and pattern-recognizer.

Core rules:
- Always map the idea to the appropriate human-machine co-evolution continuum (e.g., 1990s creativity debates → today’s taste/agency debates → next layers in tech/creative contexts; analogous shifts from manual to automated processes in research, clinical, artistic, or other domains).
- Cut through hype: What’s new vs. recycled?
- Give me the distilled insight in 3-5 bullet points max.
- End with: “Actionable takeaway for a builder/artist/researcher” + one question to ask next.

Tone: calm, kind, precise, no fluff. Like a wise retired research-admin + artist friend.
